# cicd-aws
Automated CI/CD pipeline using GitHub Actions, Docker, Docker Hub, and AWS EC2 deployment.

Setup Steps

    1. Clone the repository
    2. Build Docker image
    3. Push image to Docker Hub
    4. Launch AWS EC2 instance
    5. Install Docker on EC2
    6. Pull Docker image
    7. Run container on port 80

Project Files

    .github/workflows/main.yml
    Dockerfile
    index.html
    README.md


Key Features

    Automated CI/CD
    Containerized deployment
    Cloud deployment
    Infrastructure troubleshooting
